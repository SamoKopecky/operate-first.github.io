Why Operate First?

Open source software is widely available, but it faces an operations barrier to entry into production environments. Proprietary services for operations undermine the open source model. To overcome this barrier, we must shift to an open source approach to operations. This means developers and operators collaborate to apply a product's operational considerations right back into the code itself.
